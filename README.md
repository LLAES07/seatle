# seatle PROYECTO

| Nombre de la columna                                                 | Tipo de dato | Descripción                                                                                                                                                               |
|----------------------------------------------------------------------|--------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **OSEBuildingID**                                                    | Texto        | Identificador único asignado a cada propiedad cubierta por la Ordenanza de Benchmarking de Seattle para seguimiento e identificación.                                     |
| **DataYear**                                                         | Número       | Año al que corresponden los datos reportados.                                                                                                                             |
| **BuildingType**                                                     | Texto        | Tipo de edificio (por ejemplo, residencial, comercial).                                                                                                                   |
| **PrimaryPropertyType**                                              | Texto        | Tipo principal de propiedad del edificio.                                                                                                                                 |
| **PropertyName**                                                     | Texto        | Nombre de la propiedad o edificio.                                                                                                                                        |
| **Address**                                                          | Texto        | Dirección física del edificio.                                                                                                                                            |
| **City**                                                             | Texto        | Ciudad donde se ubica el edificio.                                                                                                                                        |
| **State**                                                            | Texto        | Estado donde se ubica el edificio.                                                                                                                                        |
| **ZipCode**                                                          | Texto        | Código postal de la ubicación del edificio.                                                                                                                               |
| **TaxParcelIdentificationNumber**                                    | Texto        | Número de identificación de parcela fiscal asignado al edificio.                                                                                                          |
| **CouncilDistrictCode**                                              | Número       | Código del distrito del consejo municipal en el que se encuentra el edificio.                                                                                             |
| **Neighborhood**                                                     | Texto        | Barrio donde se ubica el edificio.                                                                                                                                        |
| **Latitude**                                                         | Número       | Coordenada de latitud de la ubicación del edificio.                                                                                                                       |
| **Longitude**                                                        | Número       | Coordenada de longitud de la ubicación del edificio.                                                                                                                      |
| **YearBuilt**                                                        | Número       | Año en que se construyó el edificio.                                                                                                                                      |
| **NumberofBuildings**                                                | Número       | Número de edificios en la propiedad.                                                                                                                                      |
| **NumberofFloors**                                                   | Número       | Número de pisos del edificio.                                                                                                                                             |
| **PropertyGFATotal**                                                 | Número       | Área total en pies cuadrados de la propiedad.                                                                                                                             |
| **PropertyGFAParking**                                               | Número       | Área en pies cuadrados destinada a estacionamiento.                                                                                                                       |
| **PropertyGFABuilding(s)**                                           | Número       | Área en pies cuadrados de los edificios, excluyendo el estacionamiento.                                                                                                   |
| **ListOfAllPropertyUseTypes**                                        | Texto        | Lista de todos los tipos de uso de la propiedad presentes en el edificio.                                                                                                 |
| **LargestPropertyUseType**                                           | Texto        | Tipo de uso de propiedad más grande en términos de área.                                                                                                                  |
| **LargestPropertyUseTypeGFA**                                        | Número       | Área en pies cuadrados del tipo de uso de propiedad más grande.                                                                                                           |
| **SecondLargestPropertyUseType**                                     | Texto        | Segundo tipo de uso de propiedad más grande.                                                                                                                              |
| **SecondLargestPropertyUseTypeGFA**                                  | Número       | Área en pies cuadrados del segundo tipo de uso de propiedad más grande.                                                                                                   |
| **ThirdLargestPropertyUseType**                                      | Texto        | Tercer tipo de uso de propiedad más grande.                                                                                                                               |
| **ThirdLargestPropertyUseTypeGFA**                                   | Número       | Área en pies cuadrados del tercer tipo de uso de propiedad más grande.                                                                                                    |
| **YearsENERGYSTARCertified**                                         | Texto        | Años en que el edificio recibió la certificación ENERGY STAR.                                                                                                             |
| **ENERGYSTARScore**                                                  | Número       | Puntuación ENERGY STAR del edificio, que indica su eficiencia energética en comparación con edificios similares.                                                          |
| **SiteEUI**                                                          | Número       | Índice de Uso de Energía del Sitio (kBtu/pies cuadrados).                                                                                                                 |
| **SiteEUIWN**                                                        | Número       | Índice de Uso de Energía del Sitio normalizado por condiciones climáticas (kBtu/pies cuadrados).                                                                          |
| **SourceEUI**                                                        | Número       | Índice de Uso de Energía de la Fuente (kBtu/pies cuadrados).                                                                                                              |
| **SourceEUIWN**                                                      | Número       | Índice de Uso de Energía de la Fuente normalizado por condiciones climáticas (kBtu/pies cuadrados).                                                                       |
| **SiteEnergyUse**                                                    | Número       | Uso total de energía en el sitio (kBtu).                                                                                                                                  |
| **SiteEnergyUseWN**                                                  | Número       | Uso total de energía en el sitio normalizado por condiciones climáticas (kBtu).                                                                                           |
| **SteamUse**                                                         | Número       | Uso de vapor (kBtu).                                                                                                                                                      |
| **ElectricityUse**                                                   | Número       | Uso de electricidad (kWh).                                                                                                                                                |
| **ElectricityUseGridPurchase**                                       | Número       | Compra de electricidad de la red (kWh).                                                                                                                                   |
| **ElectricityUseGeneratedFromOnsiteRenewableSystems**                | Número       | Electricidad generada a partir de sistemas renovables en el sitio (kWh).                                                                                                  |
| **NaturalGasUse**                                                    | Número       | Uso de gas natural (termias).                                                                                                                                             |
| **OtherFuelUse**                                                     | Número       | Uso de otros combustibles (kBtu).                                                                                                                                         |
| **TotalGHGEmissions**                                                | Número       | Emisiones totales de gases de efecto invernadero (MtCO2e).                                                                                                                |
| **GHGEmissionsIntensity**                                            | Número       | Intensidad de emisiones de gases de efecto invernadero (kgCO2e/pies cuadrados).                                                                                           |
| **DefaultData**                                                      | Booleano     | Indica si se utilizaron datos predeterminados en lugar de datos reales.                                                                                                   |
| **ComplianceStatus**                                                 | Texto        | Estado de cumplimiento de la propiedad con la Ordenanza de Benchmarking.                                                                                                  |
| **Outlier**                                                          | Booleano     | Indica si el edificio ha sido identificado como un valor atípico en términos de uso de energía o emisiones.                                                               |
